Changing Font

Investon uses a Font family: 'Inter' as a default font from the Google Fonts Library. To change the Font, you will need to Edit the Bottom Link with your Custom Font link in main.css file and change the body css font-family

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

                                @import url('https://fonts.googleapis.com/css2?family=Lato:wght@100&display=swap');
                            
                                body {
                                    font-family: 'Inter', sans-serif
                                }
